১. utf-8 আসলে কী?

utf-8 হলো একটি টেক্সট এনকোডিং সিস্টেম।

তুমি যখন Python-এ একটা string লেখো (যেমন: "hello" বা "আমারপাসওয়ার্ড"), সেটা মূলত ইউনিকোড (Unicode) ফরম্যাটে থাকে।

কিন্তু কম্পিউটার অনেক সময় শুধু bytes (বাইট) বুঝে। তাই ইউনিকোড string → bytes এ রূপান্তর করতে হয় — সেটা করতেই .encode('utf-8') ব্যবহার করা হয়।
======================================================================================


২. কেন .encode('utf-8') ব্যবহার করি?
🔐 কারণ ১: ক্রিপ্টোগ্রাফি লাইব্রেরি bytes চায়

যেমন bcrypt বা hashlib — এরা string নেয় না, বরং bytes নেয়।
তাই আমরা এই লাইনে: password_bytes = pgen.encode('utf-8')
ইউজারনেম+পাসওয়ার্ডের কম্বাইন করা string-কে utf-8 দিয়ে bytes বানিয়ে দিচ্ছি।
========================================================================================


৩. তাহলে utf-8 কেন? আর কিছু তো দিতে পারতাম?

তুমি ঠিক ধরেছো! utf-8 ছাড়াও অন্য এনকোডিং আছে: যেমন ascii, latin-1, utf-16...

তবে utf-8 সবচেয়ে ভালো চয়েস কারণ:
| কারণ                      | ব্যাখ্যা                                      |
| ------------------------- | --------------------------------------------- |
| ✅ **সব ভাষা সাপোর্ট করে** | বাংলা, ইংরেজি, চীনা, ইমোজি সব এনকোড করতে পারে |
| ✅ **সবচেয়ে জনপ্রিয়**      | ওয়েব ও প্রোগ্রামিং জগতে `utf-8` স্ট্যান্ডার্ড |
| ✅ **কম স্পেস নেয়**        | ইংরেজি অক্ষর ১ বাইটেই হয়                      |
| ✅ **ডিটারমিনিস্টিক**      | একই `string` → একই `bytes` বারবার             |
=======================================================================================


৪. সমস্যা হতে পারে?
হ্যাঁ, যদি encode না করো বা ভুল encode করো, তাহলে...

🔴 bcrypt.hashpw() এরর দিবে: "TypeError: Unicode-objects must be encoded before hashing"

🔴 বিভিন্ন সিস্টেমে একই পাসওয়ার্ড ভিন্ন bytes হতে পারে যদি একই এনকোডিং না হয়

🔴 কিছু Unicode ক্যারেক্টারের একাধিক রূপ থাকতে পারে (যেমন: é vs e + ´)

✅ তাই আমরা নির্ভরযোগ্যভাবে ও স্ট্যান্ডার্ডভাবে utf-8 ব্যবহার করি।
======================================================================================

UTF-8 হল একটি টেক্সট এনকোডিং ফরম্যাট, যা ইউনিকোড (Unicode) ক্যারেক্টারগুলোকে বাইনারি (bytes) তে রূপান্তর করে।

"Hello" (unicode character) ============> bcrypt library (error)
"Hello" (unicode character) ============> Convert to Bytes with utf-8 ==============> bcrypt library (ok)
